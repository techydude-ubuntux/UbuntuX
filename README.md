# UbuntuX---Minimal-Termux-Ubuntu
UbuntuX Lite is a lightweight and smooth Ubuntu environment for Android built using Termux, Termux:X11, and Proot-Distro.  This project was created by **Shivansh Joshi (14 years old) with the goal of making Linux and programming accessible to students and users who cannot afford a PC or laptop and can learn programming from their phones only.
